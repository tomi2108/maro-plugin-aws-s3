# 🌩️ maro-plugin-aws-s3

Interpreta los susurros de las nubes.

Este plugin extiende Maro para abrir los baúles secretos de Amazon AWS S3.  
Permite listar, subir, borrar archivos y consultar metadatos en buckets S3, todo desde la terminal, sin perderse en el vasto firmamento.

Un conjuro ideal para quienes viajan entre clústeres y almacenamientos distribuidos sin perder el rumbo ni la seguridad.

---
**Comandos principales**
- Listar archivos en un bucket.
- Consultar metadatos de objetos.
- Subir archivos a la nube.
- Eliminar contenido con precisión.

Usa este círculo para custodiar, mover y descifrar los tesoros digitales dispersos por Amazon.
